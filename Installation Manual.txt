Python Install :
    1) Python 3.8.0 (becase it is developed in this version, you can download higher but with no guarantee of compatibility)
        - link: https://www.python.org/downloads/release/python-380/
        - install it as "Run as Administrator"
        - on pop-up page mark "Add Python 3.8 to PATH" and unmark "Instal launcher for all users" (if possible)

Library installation:
    1) run command line (cmd)
    2) in powershell write "cd C:\Users\...\AppData\Local\Programs\Python\Python38\Scripts\" --> path to the pip.exe file 
    3) write all these lines one by one and hit enter:
        pip install pandas
        pip install pyodbc
        pip install openpyxl
        pip install tqdm
